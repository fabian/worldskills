Website for Skills/Compétences Canada
======================================

This project has been developed by Fabian Vogler during the 
WorldSkills 2009 competition in Calgary, CA. While working on the 
project itself any kind of communication and access to private 
documentation or the internet was not allowed. The project had
to be completed within 22 hours, spread over four days.

Requirements: PHP5 with PDO and XSL, MySQL
